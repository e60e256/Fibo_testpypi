## fibo_testpypi
This is a test library to upload on test.pypi.org.
It has one class and one method in it.

- Fibo
The class.
- Fibo.fibo(x)
Calculate x th fibonacci number. x must be a positive interger.


### Usage
```python
import fibo_testpypi
Fibo = fibo_testpypi.Fibo()
print(Fibo.fibo(1)) # 1
print(Fibo.fibo(3)) # 2
print(Fibo.fibo(5)) # 5
